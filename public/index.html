Link to API


Endpoints:

Endpoints HTTP Method Response Data Type Description
/students GET [] This endpoint returns an array of all the students stored within the API
/students/:id GET {} This endpoint returns Object representing only 1 student
/students POST {} This endpoint is used to create a new student. It will return an object representing the new student that was created
/students/:id PUT {} This endpoint is used for updating an already exisiting student. The id in the param will represent the id of the student we want to update the value of
/students/:id DELETE {} This endpoint is used for deleting an exisiting students data. it returns an empty object
